# Diagramas del sistema

En esta carpeta se encuentran los diagramas de flujo desarrollados durante el proceso del proyecto, aplicando el modelo de construccion de prototipo.

## Versiones incluidas

- **Prototipo 1:** Ingreso de datos (nota y asistencia)
- **Prototipo 2:** Evaluación de nota menor a 3.0
- **Prototipo 3:** Inclusión del rango entre 3.0 y 3.9
- **Prototipo 4:** Lógica completa del sistema
- **Diagrama final:** Versión optimizada con validación de datos

## Descripción

Cada diagrama representa una mejora progresiva del sistema, permitiendo evidenciar la evolución del algoritmo hasta su versión final.

Estos diagramas fueron utilizados para modelar la lógica del programa antes de su implementación.
