# Taller Final - Condicionales y Modelos de Desarrollo de Software

## Descripción
Este proyecto consiste en el desarrollo de un algoritmo que determina el estado académico de un estudiante según su nota final y porcentaje de asistencia, utilizando estructuras condicionales.

---

## Modelo de desarrollo aplicado
Se utilizó el  Modelo de construcción de prototipo, el cual permite construir soluciones de forma progresiva mediante versiones iniciales que se van mejorando hasta llegar a una versión final funcional.

---

## Evolución del sistema (Prototipos)

- 🔹 Prototipo 1: Ingreso de datos (nota y asistencia)
- 🔹 Prototipo 2: Evaluación de nota menor a 3.0
- 🔹 Prototipo 3: Inclusión del rango entre 3.0 y 3.9
- 🔹 Prototipo 4: Implementación completa de condiciones
- 🔹 Versión final: Validación de datos y optimización del algoritmo
