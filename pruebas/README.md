# Pruebas del sistema

En esta carpeta se encuentran las pruebas realizadas al algoritmo para verificar su correcto funcionamiento.

## Descripción

Se evaluaron diferentes combinaciones de valores de nota y asistencia para validar cada uno de los posibles resultados del sistema.

## Objetivo

Comprobar que el algoritmo:

- Evalúa correctamente las condiciones
- Responde adecuadamente a cada caso
- Maneja valores fuera de rango

## Archivo

- **pruebas.txt**: Contiene los casos de prueba y sus respectivos resultados
