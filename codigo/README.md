# Código fuente

En esta carpeta se encuentra el código fuente del algoritmo desarrollado en PSeInt.

## Archivo incluido

- **algoritmo_final.psc**: Contiene la versión final del algoritmo que determina el estado académico de un estudiante según su nota y porcentaje de asistencia.

## Descripción

El algoritmo utiliza estructuras condicionales para evaluar diferentes rangos de nota y asistencia, determinando si el estudiante:

- Pierde por nota
- Pierde por inasistencia
- Pierde por ambas
- Aprueba
- Queda en revisión académica
- Aprueba con buen desempeño

## Uso

El archivo puede ejecutarse en el entorno PSeInt para verificar su funcionamiento.
